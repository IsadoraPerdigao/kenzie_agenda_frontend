# kenzie_agenda_frontend
A Kenzie Agenda simula uma agenda desenvolvida em React.JS.

## Instalando dependências
```bash
# Caso use npm
npm run i

# Caso use yarn
yarn
```

## Rodando a aplicação localmente
```bash
# caso use npm
npm run dev

# caso use yarn
yarn dev
```
